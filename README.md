# dl-card-creator

## Usage

1. serial.svg をつくる．
  - シリアルナンバーの部分は ``AABBCCDDEEFF`` にする
  - カードNo. の部分は ``No.001`` にする
2. [対面電書][densyo]から，シリアルとカード No. を手に入れる
  - ![](https://gyazo.com/e5e6b38ef4f8bd19e0eec70e764b4c5a.png)
2. ``npm i && node ./create.js``
3. おわり
